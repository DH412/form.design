Login Page

The Login Page allows the users to login. It has two fields, the User Name, and the Password. The inputs will be compared with our User database and the password verified. Once the verification complete, the user will proceed to the Main Menu. If the user is new, s/he should click on the "Join Now" link to provide the necessary information and be added to the database. If the user wants to delete another user, s/he should click on the "For the admin only" link.
